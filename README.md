# 🌿 AgroAsesor – Panel de Empleados

Panel web interno para asesoría de ventas agrícolas con inteligencia artificial.

## ¿Qué hace?
- **Asesor IA** conectado al inventario de 228 productos agrícolas
- **Registro de ventas** con autocompletado de productos
- **Recordatorios de seguimiento** automáticos vía Google Calendar
- **Historial de experiencias** de productores locales
- **Chat inteligente** que recomienda alternativas del inventario propio

## Tecnologías
- Frontend: HTML + CSS + JS (sin dependencias)
- Backend: n8n Cloud (workflows automatizados)
- Base de datos: Airtable
- IA: Claude Sonnet (Anthropic)
- Calendario: Google Calendar API

## Configuración
1. Abre el panel en tu navegador
2. Ve a ⚙️ Configuración
3. Pega las URLs de los webhooks de n8n
4. Guarda y listo

## Uso exclusivo interno
Este panel está diseñado para uso exclusivo de los empleados de la tienda.
